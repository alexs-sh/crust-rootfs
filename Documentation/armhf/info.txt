Описание
========

Образ КФС Debian Installer полученный из офф. ISO-образа. Оформлен в виде
initramfs, который может быть загружен в устройство при помощи u-boot

После запуска позволяет установить Debian на устройство при помощи его штатного
установщика

Загрузчик и ядро(armmp) Debian не работает на устройствах imx6ull. Поэтому 
их следует получить(собрать, скачать) отдельно. Раздельное ядро и загрузчик
приводят к тому, что:
  1. важно не записать Debian/его компоненты в раздел, где находятся
  пользовательские ядра 
  2. ядро Debian установленное в /boot не используется
  3. драйвера полученные через apt не будут работать, т.к. рассчитаны на другую
  версию ядра. 

Хорошим вариантом разметки явпяется:
раздел №1 - FAT32 - содержит пользовательсие загрузчик, ядро, dtb
раздел №2 - EXT2/3/4 - содержит Debian. Debian не знает о существовании раздела
№1 (раздел никак не монтируеся)


Пример конфигурации U-Boot
==========================
Загрузка Debian Installer (initrd.uboot)

    setenv fdt_file imx6ull-nano.dtb
    setenv initrd_file initrd.uboot
    setenv image zImage

    fatload mmc ${mmcdev}:${mmcpart} ${loadaddr} ${image}
    fatload mmc ${mmcdev}:${mmcpart} ${fdt_addr} ${fdt_file}
    fatload mmc ${mmcdev}:${mmcpart} ${initrd_addr} ${initrd_file}
    bootz ${loadaddr} ${initrd_addr} ${fdt_addr}


Особенности
===========

Консоль
-------

Netinstaller настраивает запуск консоли с возможностью выбора скорости и
поддержкой терминала vt220. Этот вариант нам не подходит. 
Исправляем исправляем /lib/systemd/system/serial-getty@.service 
Записываем фиксированное значение в аргумент запуска:
ExecStart=/sbin/agetty 115200 %I
